# Database Seeding

This folder is specifically about data sources to seed our database. Our database requires extensive information on: 

1. Airports and airport codes
2. Destinations
3. Hotels for each destination

The current sources are: 

- Airports: https://github.com/aashishvanand/airport-data-js/blob/main/data/airports.json --> `airports.json`
- Destinations: Geonames cities5000.txt and countryInfo.txt --> `destinations.json`
- Hotels: Sourced from Overpass API --> `hotels.json`
